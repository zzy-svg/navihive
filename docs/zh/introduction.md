# 项目介绍

## NaviHive 是什么？

NaviHive 是一个**现代化的个人导航站管理系统**，基于 Cloudflare Workers 构建，提供零成本部署、极致性能和企业级安全的导航站解决方案。

无论你是个人开发者、团队管理者，还是内容创作者，NaviHive 都能帮你轻松管理和分享你的网站收藏。

## 主要特点

### 💰 零成本运行

- 基于 Cloudflare Workers 免费套餐
- 每月 100,000 次免费请求
- 无需购买服务器和域名（可选）
- 永久免费使用

### ⚡ 极致性能

- 全球 300+ CDN 节点
- 毫秒级响应时间
- 自动边缘缓存
- 无需任何性能优化配置

### 🔐 企业级安全

经过 **14 个安全修复提交**，包括：

- JWT + bcrypt 认证加密
- HttpOnly Cookie 防 XSS 攻击
- SQL 注入防护（参数化查询）
- SSRF 防护（URL 验证）
- 登录速率限制（5 次/15 分钟）
- 完整的输入验证

### 🌍 访客模式

**v1.1.0 新特性**

- 支持公开/私密内容分离
- 未登录用户可浏览公开内容
- 管理员查看全部数据
- 灵活的访问控制

### 🔄 拖拽排序

- 可视化调整分组和网站顺序
- 所见即所得的交互体验
- 自动保存排序结果
- 基于 DND Kit 实现

### 🎨 高度自定义

- 自定义 CSS 样式
- 自定义背景图
- 自定义主题色
- 暗色模式支持

- **客户案例展示**：展示项目作品和案例链接

### 🎨 内容创作

- **资源收集**：整理创作素材来源

## 技术亮点

### 全栈 TypeScript

- 前后端完全类型安全
- 严格模式（Strict Mode）
- 减少运行时错误

### 现代化技术栈

- **前端**：React 19 + Material UI 7 + Tailwind CSS 4
- **后端**：Cloudflare Workers（Serverless）
- **数据库**：Cloudflare D1（SQLite）
- **构建**：Vite 6

### Serverless 架构

- 无需管理服务器
- 自动伸缩
- 按需计费（免费额度充足）
- 全球边缘部署
